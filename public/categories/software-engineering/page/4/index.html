MySQL 의 필드 type 변환함수 CAST

(애초에 테이블을 만들 때 필드를 숫자 형태로 만들었어야 한다…) CAST함수를 사용한다면, * 쿼리 : select * from prices order by CAST(price AS UNSIGNED); * 결과 : 220 280 380 450 570 760 850 1060 1420 1590 1990 2390 2660 2990 3420 3990 4270 5700 제대로 나온다.
