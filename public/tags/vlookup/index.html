Excel의 VLOOKUP 함수의 첫번째 인자의 값에 ~ (tilda 물결표시)가 있는 경우 찾기오류 발생

2007-12-12 by Dennis in  Software Engineering  VLOOKUP, excel

이런 경우 첫번째 인자에서 ~를 제거하거나 다른 문자로 대치하여야 제대로 동작한다 P2 의 값이 “서울~부산” 이라고 할 때 VLOOKUP( P2,구간코드,2,FALSE) 위의 함수를 아래 처럼 VLOOKUP( SUBSTITUTE(P2,”~“,”-“),구간코드,2,FALSE) 하고 “구간코드” 에 해당 하는 table_array의 값에도 ~ 를 - 로 대치해야한다. 링크 : http://www.mrexcel.com/archive2/41300/48504.htm -- Read more →
