IT 분야에서 ‘프레임’은 자주 사용되므로 용어자체는 낯설지 않았다. 애니메이션gif에서 하나의 장면을 ‘프레임’이라 부른다.(Adobe 플래시에서도 비슷한 의미로 쓰인다) 하나의 브라우저 창에 여러 html 파일을 보여줄 때 각각의 html 을 담고 있는 틀을 ‘프레임’ 이라 부른다. 소프트웨어 시스템의 ‘얼개’를 framework 혹은 간략히 frame 이라고 한다. 시스템을 구축 프로젝트 초기에 framework 선정의 문제는 아주 중요하기 때문에 엔지니어들 사이에서는 항상 주요 이슈이다.
