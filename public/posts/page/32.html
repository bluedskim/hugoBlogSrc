FTP script 샘플

암호 중간에 스페이스 문자가 있을 때 아래와 같이 한다. #!/bin/sh HOST=‘ftp.somehost.com’ USER=‘loginid’ PASSWD=‘pass word’ ftp -v -n $HOST <<END_SCRIPT quote USER $USER quote PASS $PASSWD passive pwd quit END_SCRIPT exit 0 --
