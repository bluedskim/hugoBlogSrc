javascript로 해당 html 내부의 모든 링크 목록 뽑아내기

November 2, 2007

아래 코드를 </body> 앞에 삽입한 후 브라우저로 불러보면 제일 아래쪽에 링크들 목록이 뜬다.

<script>
    for(i = 0 ; i < document.links.length ; i++ ) {
        document.write(’<xmp>’);
        document.write(’[’ + (i+1) + ‘]번째 링크 : ‘ + document.links[i].href + ‘ ☞ ’ + document.links[i].innerHTML);
        document.write(’</xmp>’);
    }
</script>
