2007-04-25 by Dennis in  Software Engineering

아래코드를 그대로 copy & paste 하면 됨

Enumeration em = request.getHeaderNames();

while (em.hasMoreElements()){

    String s = (String)em.nextElement();
    
    out.println(s +” : “);
    
    out.println(request.getHeader(s)+”<br>”);

}
